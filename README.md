# 🚤 Sea4U WhatsApp Bot

בוט WhatsApp חכם מבוסס בינה מלאכותית (AI) עבור Sea4U - השכרת יאכטות במרינה הרצליה.

## 🎯 מה הבוט עושה?

- ✅ עונה ללקוחות בצורה טבעית ואנושית (AI)
- ✅ דובר עברית בלבד! 🇮🇱
- ✅ עונה על כל הודעה (גם "היי", "שלום", "מה המחיר" וכו')
- ✅ אוסף מידע: תאריך, מספר אנשים, מתי נוח לחזור
- ✅ עובד בשעות פעילות בלבד (7:00-21:00)
- ✅ לא עובד בשבת (שומר שבת!)
- ✅ כתוב בעברית מקצועית וחברותית

## 📂 מבנה הפרויקט

```
whatsapp_bot/
├── app.py                 # קוד הבוט הראשי
├── requirements.txt       # תלויות Python
├── Procfile              # הגדרות Railway
├── runtime.txt           # גרסת Python
├── .env.example          # דוגמה למשתני סביבה
├── SETUP_GUIDE.md        # מדריך התקנה מפורט
└── README.md             # הקובץ הזה
```

## 🚀 התקנה מהירה

1. **קרא את [SETUP_GUIDE.md](SETUP_GUIDE.md)** - יש שם הכל!
2. השג מפתחות API (OpenAI + Meta)
3. העלה ל-Railway
4. חבר ל-WhatsApp
5. זהו! 🎉

## 🔑 משתני סביבה נדרשים

```env
OPENAI_API_KEY=sk-...
WHATSAPP_TOKEN=EAA...
PHONE_NUMBER_ID=123456789
VERIFY_TOKEN=sea4u_verify_token_2024
PORT=5000
```

## 📱 איך זה עובד?

1. לקוח שולח הודעה ב-WhatsApp ל-077-2310890
2. הבוט מקבל את ההודעה דרך Webhook
3. הבוט שולח את ההודעה ל-OpenAI (GPT-4)
4. AI מחזיר תשובה חכמה ואנושית **בעברית**
5. הבוט שולח את התשובה ללקוח

### דוגמה לשיחה:
```
לקוח: היי
בוט: שלום! איך נוכל לעזור לך? 😊

לקוח: כמה עולה שייט?
בוט: המחירים שלנו מתחילים מ-550 ₪ לזוג ועד 1,300 ₪ לקבוצה של 13 אנשים. 
לאיזה תאריך אתם מתכננים?

לקוח: 15 לחודש הבא, 8 אנשים
בוט: מעולה! 8 אנשים זה מושלם. מה האירוע? יום הולדת? אירוע חברה?

לקוח: יום הולדת!
בוט: כמה כיף! 🎉 מתי הכי נוח לכם שחזי יתקשר לתאם את כל הפרטים?
```

## ⏰ שעות פעילות

- **ימים רגילים:** 07:00 - 21:00
- **שבת:** לא פעיל (מיום שישי 18:00 עד מוצ"ש 20:30)

## 💻 טכנולוגיות

- **Python 3.11**
- **Flask** - Web server
- **OpenAI GPT-4** - בינה מלאכותית
- **WhatsApp Business API** - חיבור ל-WhatsApp
- **Railway** - Hosting

## 📊 עלויות חודשיות

- Railway: ~$5-10
- OpenAI: ~$10-20
- WhatsApp: חינם (עד 1000 הודעות)
- **סה"כ:** ~$15-30/חודש

## 🔧 פיתוח מקומי

```bash
# התקנת תלויות
pip install -r requirements.txt

# הגדרת משתני סביבה
cp .env.example .env
# ערוך את .env עם המפתחות שלך

# הרצת השרת
python app.py
```
